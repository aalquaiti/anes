# anes
A Work-In-Progress NES Emulator in Java



## Progress

### CPU
The 6502 CPU official opcodes are fully supported, with most of the import unofficial opcodes implemented as well

### PPU
Not implemented yet

### APU
Not implemented yet

### Mappers
Supported mappers include:
- Mapper0
